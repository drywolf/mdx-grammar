Generation

Returns the generation of the input member.

Syntax

member.Generation
ParameterDescription

member

Member specification.

Example

The following query

SELECT
 [Year].[Qtr1].Generation.Members
ON COLUMNS,
 [Product].Generations(2).Members
ON ROWS
FROM Sample.Basic

returns the grid:
