InStr

Returns a number specifying the position of the first occurrence of one string within another.

Syntax

InStr ( [start,] string1, string2 [,compare] )
ParameterDescription

start

Optional. Character position to begin search. If omitted, search begins at first character in string1.

string1

String in which to search (or field containing that string).

string2

String for which to search (or field containing that string).

compare

Optional search mode. Values: 0 for case sensitive, 1 for case insensitive. Default is case sensitive.

Example

InStr (5, Year2000_promotional, promotional, 1)

returns 10
