Attribute

Returns all base members that are associated with a specified attribute member.

Syntax

Attribute ( member )
ParameterDescription

member

Specification of a member from an attribute dimension.

Example

The following query

SELECT
 {[Year].Children}
ON COLUMNS, 
 Attribute ([Ounces_12])
ON ROWS
FROM Sample.Basic

returns the grid:
